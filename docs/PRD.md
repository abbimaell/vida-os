# PRD

## VIDA OS v0.1.0

VIDA OS v0.1.0 delivers the first working path for capturing Radiografia de Formacion assessments in WordPress.

## Scope

- WordPress plugin foundation.
- Database tables for people and assessments.
- Internal service for saving assessments.
- REST endpoint for frontend submissions.
- Frontend integration without Firebase.

## Out of Scope

- Dashboard Pastoral.
- Person profile views.
- Growth history.
- Mentoring workflows.
- Reports.
- Exports.
- Authentication flows.
