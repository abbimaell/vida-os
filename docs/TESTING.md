# TESTING

Use this checklist to validate VIDA OS v0.1.0.

- [ ] Plugin instala correctamente.
- [ ] Plugin activa correctamente.
- [ ] Tablas creadas.
- [ ] Namespace `vida-os/v1` visible.
- [ ] Endpoint POST responde.
- [ ] Evaluacion se guarda.
- [ ] Persona se crea.
- [ ] Evaluacion se relaciona con la persona.
- [ ] Frontend muestra resultados.
- [ ] Frontend envia `email`.
- [ ] Frontend envia `display_name`.
- [ ] Frontend envia `evaluation_period`.
- [ ] Frontend envia `responses`.
- [ ] Frontend no contiene referencias a Firebase.
